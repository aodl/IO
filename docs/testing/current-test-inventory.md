Explicit current gaps:

- real SNS neuron staking/top-up/minimum-stake and dissolve-delay visibility are proved only in direct real governance installs, not through SNS-W finalization.
- real SNS governance maturity/rewards are not proved.
- real SNS governance voting/following is not proved.
- full SNS-W publication is not proved because `sns_governance.wasm` cannot be submitted through direct PocketIC update or direct `manage_neuron` proposal ingress; the captured proposal request is 6,724,190 bytes and the max ingress is 3,670,016 bytes, so the next driver must avoid one large ingress message.
- real SNS index account history is proved only by opt-in `e2e-real-canisters` ignored tests when pinned SNS ledger/index Wasms are supplied; it is not proved in default CI.
- real SNS root is proved only for a direct-install list/dapp-registration smoke, not for finalized-SNS control.
- IO APY increase from real SNS staked IO is not proved.
- full ICP -> IO -> stake -> APY -> redemption E2E with real SNS ledger/index/governance/root is not proved.


## Current Real-Framework Stride

`tools/scripts/run-real-framework-e2e` is the opt-in local operator path for pinned real framework artifacts. It fetches/verifies/decompresses configured artifacts, runs the real ledger/index tests, the PocketIC ICP-feature SNS-W bootstrap proof, gzipped NNS proposal publication for all six SNS Wasm slots, SNS-W deploy/swap-open tests, and the real-ledger exact-economics E2E. Governance/root/SNS-W normal staking remains blocked until swap participation/finalization and list-neurons drivers are implemented.
